# LeetMath

A web platform for grinding university-level math the way coders grind LeetCode — a **Practice** track of instantly-validated problems and a **Learn** track of structured courses, wrapped in an XP / rank progression system with daily challenges and streaks.

Built with **Next.js** (App Router), **React**, **TypeScript**, **Tailwind CSS**, and **Supabase** (Postgres + Auth + RLS), with **KaTeX** for math.

> In active development — launching soon.

---

## What it is

LeetMath turns learning advanced math into a practice habit, spanning **Mathematics**, **Artificial Intelligence**, and **Physics & Astronomy**. Solve problems with instant, server-side validation; work through structured courses; and progress through an XP and rank system with daily challenges and streaks.

---

## Features

- **Practice** — LeetCode-style math problems with instant, server-side answer validation (number, expression, matrix, vector, multiple-choice, text), multi-step problems, and four difficulty tiers.
- **Learn** — a courses › boxes › lessons content hierarchy with rich lessons and in-lesson comprehension questions.
- **Gamification** — XP, ~30 rank tiers, a daily challenge, and UTC-day streaks.
- **Dashboard** — progress, stats, activity heatmap, and "continue learning".
