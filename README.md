# .github

This repository holds organization-wide defaults for **[leetmath-app](https://github.com/leetmath-app)**.

## Contents

- **`profile/README.md`** — the public profile shown on the organization's page.
- Shared community-health files (issue / pull request templates, contributing
  guidelines, etc.) that apply as defaults across all repositories in the org.

The application source code lives in the private
[`leetmath-app`](https://github.com/leetmath-app/leetmath-app) repository.
